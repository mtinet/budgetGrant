# 교부확인서 발급 시스템 사용 가이드

---

## 🏫 사용자 페이지

학교 담당자가 교부확인서를 발급받는 페이지입니다.

1. **학교명**과 **NEIS 번호**를 입력합니다.
2. **교부확인서 출력** 버튼을 클릭합니다.
3. 인쇄 창에서 **대상: PDF로 저장**을 선택하여 저장합니다.

> - 발급 기간이 아닌 경우 자동으로 접근이 차단됩니다.
> - 학교명과 NEIS 번호가 정확히 일치해야 발급이 가능합니다.

---

## 👩‍💼 관리자 페이지

### 로그인
관리자 비밀번호를 입력하여 로그인합니다. 로그인 후 10분간 세션이 유지됩니다.

---

### 담당자 전화번호 설정
사용자 페이지와 교부확인서에 표시되는 담당자 전화번호를 변경합니다.

1. 변경할 전화번호를 입력합니다.
2. **전화번호 저장** 버튼을 클릭합니다.

---

### 발급 기간 설정
학교에서 교부확인서를 발급받을 수 있는 기간을 설정합니다.

1. 시작일과 시작 시간을 선택합니다.
2. 종료일과 종료 시간을 선택합니다.
3. **발급 기간 저장** 버튼을 클릭합니다.

---

### 데이터 업로드
교부확인서에 들어갈 학교별 데이터를 엑셀로 업로드합니다.

1. **샘플 엑셀 다운로드**로 양식을 받아 데이터를 입력합니다.

| 학교명 | NEIS번호 | 사업명 | 교부금액 | 교부일 |
|---|---|---|---|---|
| 서울○○고등학교 | B100000001 | 2026년 글로벌 현장학습 | 5000000 | 2026-04-01 |

2. 작성한 엑셀 파일을 업로드 영역에 드래그앤드롭하거나 클릭하여 선택합니다.
3. 미리보기로 데이터를 확인한 후 **DB에 저장하기** 버튼을 클릭합니다.

> 교부금액은 숫자만 입력하세요. 자동으로 `5,000,000` 형식으로 변환됩니다.

---

### 일괄 출력
DB에 등록된 전체 학교의 교부확인서를 한 번에 출력합니다.

1. **전체 교부확인서 일괄 출력** 버튼을 클릭합니다.
2. 인쇄 창에서 **대상: PDF로 저장**을 선택하여 저장합니다.
3. 인쇄 완료 후 창이 자동으로 닫힙니다.

---

### DB 관리
**DB 데이터 즉시 폐기** 버튼으로 등록된 전체 데이터를 삭제합니다.
